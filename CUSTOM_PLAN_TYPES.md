# 🎨 Создание собственных типов тарифных планов

## ✅ Реализовано

Теперь вы можете **создавать свои собственные типы** тарифных планов, а не только выбирать из предустановленных (FREE, PRO, PREMIUM).

## 🚀 Как использовать

### Создание плана с собственным типом

1. **Откройте страницу создания плана:**
   ```
   http://localhost:3000/admin/subscription-plans/new
   ```

2. **Выберите "Создать свой тип"** (радио-кнопка)

3. **Введите название типа:**
   - Например: `VIP`, `Gold`, `Silver`, `Student`, `Family`
   - Можно использовать пробелы: `Super Premium`
   - Можно использовать кириллицу: `Золотой`

4. **Система автоматически преобразует:**
   - `VIP` → `VIP`
   - `Gold Plan` → `GOLD_PLAN`
   - `Super Premium` → `SUPER_PREMIUM`
   - `Золотой` → `ЗОЛОТОЙ`

5. **Заполните остальные поля** (названия, описания, цены, функции)

6. **Нажмите "Создать план"**

## 🎯 Правила именования

### Разрешено:
- ✅ Латинские буквы: `VIP`, `GOLD`, `SILVER`
- ✅ Кириллица: `ЗОЛОТОЙ`, `СЕРЕБРЯНЫЙ`
- ✅ Цифры: `PLAN2024`, `LEVEL5`
- ✅ Пробелы (будут заменены на `_`): `Super Premium` → `SUPER_PREMIUM`
- ✅ Дефисы: `PRO-PLUS` → `PRO-PLUS`

### Автоматические преобразования:
- Все буквы → ВЕРХНИЙ РЕГИСТР
- Пробелы → подчеркивания (`_`)
- Множественные пробелы → один `_`
- Начальные/конечные пробелы удаляются

### Примеры преобразований:
```
Ввод              → Результат
"vip"             → "VIP"
"Gold Plan"       → "GOLD_PLAN"
"super  premium"  → "SUPER_PREMIUM"
"  starter  "     → "STARTER"
"Pro-Plus"        → "PRO-PLUS"
"Золотой"         → "ЗОЛОТОЙ"
```

## ⚠️ Ограничения

### Уникальность
- Тип плана должен быть **уникальным**
- Нельзя создать два плана с одинаковым типом
- Если план с таким типом существует, появится ошибка

### Длина
- Минимум: **2 символа**
- Максимум: **50 символов**

### Запрещено:
- ❌ Пустое значение
- ❌ Только пробелы
- ❌ Специальные символы: `@`, `#`, `$`, `%`, `&`, `*`

## 🔄 Миграция

### Что изменилось:
1. **Тип `plan_type` изменен с enum на varchar(50)**
   - Раньше: только FREE, PRO, PREMIUM
   - Теперь: любые значения

2. **Существующие планы сохранены**
   - FREE → "FREE"
   - PRO → "PRO"
   - PREMIUM → "PREMIUM"

3. **Обратная совместимость**
   - Все существующие планы работают
   - Можно создавать новые типы
   - Можно использовать старые типы
