# 🎨 Современный блочный лендинг Bite Lens

## ✅ Что создано

Полностью функциональный современный лендинг с блочным дизайном для приложения Bite Lens.

## 🎯 Основные секции

### 1. **Header (Шапка)**
- Фиксированная навигация с эффектом blur
- Логотип с градиентом
- Навигационное меню
- Кнопки "Войти" и "Начать"

### 2. **Hero Section (Главный экран)**
- Крупный заголовок с градиентным текстом
- Описание продукта
- Две CTA кнопки
- Социальное доказательство (аватары пользователей + рейтинг)
- Интерактивная карточка с примером анализа еды

### 3. **Features (Возможности)**
- 6 карточек с ключевыми функциями
- Иконки-эмодзи
- Hover эффекты
- Адаптивная сетка

### 4. **How It Works (Как работает)**
- 3 шага с нумерацией
- Градиентные карточки
- Стрелки между шагами

### 5. **Stats (Статистика)**
- Яркая секция с градиентным фоном
- 4 ключевых метрики
- Крупные цифры

### 6. **Pricing (Тарифы)**
- 3 тарифных плана
- Выделенный популярный план
- Список возможностей
- CTA кнопки

### 7. **CTA Section (Призыв к действию)**
- Финальный призыв
- Градиентный фон
- Две кнопки действия

### 8. **Footer (Подвал)**
- Логотип и описание
- 4 колонки с ссылками
- Социальные сети
- Copyright

## 🎨 Дизайн-система

### Цвета
- **Основной градиент**: Emerald (зеленый) → Teal (бирюзовый)
- **Светлая тема**: Белый фон, серые акценты
- **Темная тема**: Черный фон, серые акценты

### Типографика
- **Заголовки**: Bold, крупные размеры (4xl-6xl)
- **Текст**: Regular, читаемые размеры (lg-xl)
- **Шрифт**: Geist Sans (системный fallback)

### Компоненты
- **Кнопки**: Rounded-full (полностью скругленные)
- **Карточки**: Rounded-2xl/3xl с тенями
- **Hover эффекты**: Transform scale, тени, цвета

## 🌓 Dark Mode

Полная поддержка темной темы:
- Автоматическое переключение по системным настройкам
- Все секции адаптированы
- Правильные контрасты

## 📱 Адаптивность

Полностью адаптивный дизайн:
- **Mobile**: 1 колонка, вертикальные стеки
- **Tablet**: 2 колонки
- **Desktop**: 3-4 колонки, горизонтальные раскладки

Breakpoints:
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px

## ✨ Интерактивность

### Анимации
- Smooth scroll для якорных ссылок
- Hover эффекты на карточках
- Transform scale на кнопках
- Fade-in анимации (готовы к использованию)

### Эффекты
- Backdrop blur на header
- Градиенты на кнопках и секциях
- Тени при hover
- Плавные переходы

## 🚀 Как запустить

```bash
# Установите зависимости (если еще не установлены)
npm install

# Запустите dev сервер
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 🔧 Кастомизация

### Изменить цвета
В `app/page.tsx` замените классы:
- `from-emerald-500 to-teal-600` → ваш градиент
- `emerald-*` → ваш цвет
- `teal-*` → ваш цвет

### Изменить контент
Все тексты находятся в `app/page.tsx`:
- Заголовки
- Описания
- Списки возможностей
- Тарифы

### Добавить секции
Скопируйте структуру существующей секции и адаптируйте под свои нужды.
