# 👥 Система управления пользователями

## Дата создания: 4 мая 2026

## 📋 Обзор

Полноценная система управления пользователями для админ-панели Bite Lens с таблицей, статистикой, поиском, фильтрацией и модальным редактированием.

---

## 🎯 Реализованные функции

### 1. **Статистика пользователей**
Четыре информативные карточки:
- **Всего пользователей** - общее количество USER
- **Новых за 24ч** - регистрации за последние сутки
- **Активных сегодня** - количество активных сессий сегодня
- **Администраторов** - количество ADMIN

### 2. **Таблица пользователей**
Колонки:
- **Пользователь** - аватар (инициалы), имя, email
- **Роль** - USER или ADMIN с иконками
- **Email** - статус подтверждения (✓ Подтвержден / Не подтвержден)
- **2FA** - статус двухфакторной аутентификации
- **Дата регистрации** - дата создания аккаунта
- **Действия** - редактирование и удаление

### 3. **Поиск и фильтрация**
- **Поиск** - по email или имени (регистронезависимый)
- **Фильтр по роли** - все / пользователи / администраторы
- **Пагинация** - по 10 пользователей на страницу

### 4. **Модальное окно редактирования**
- Просмотр информации о пользователе
- Изменение роли (USER ↔ ADMIN)
- Изменение статуса подтверждения email
- Защита от изменения собственной роли

### 5. **Удаление пользователей**
- Двухэтапное подтверждение (иконки)
- Защита от удаления себя
- Автоматическое обновление списка

## 🔧 Технические детали

### API Endpoints

#### `GET /api/admin/users`
Получение списка пользователей с пагинацией

**Query параметры:**
- `page` - номер страницы (default: 1)
- `limit` - количество на странице (default: 10)
- `search` - поиск по email/имени
- `role` - фильтр по роли (USER/ADMIN)

**Response:**
```json
{
  "users": [
    {
      "id": "uuid",
      "email": "user@example.com",
      "name": "John Doe",
      "role": "USER",
      "email_verified": true,
      "two_factor_enabled": false,
      "created_at": "2026-05-04T10:00:00.000Z",
      "updated_at": "2026-05-04T10:00:00.000Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 50,
    "totalPages": 5
  }
}
```

#### `GET /api/admin/users/stats`
Получение статистики пользователей

**Response:**
```json
{
  "totalUsers": 100,
  "newUsersLast24h": 5,
  "activeToday": 25,
  "totalAdmins": 3,
  "verifiedUsers": 80
}
```

#### `GET /api/admin/users/[id]`
Получение информации о конкретном пользователе

#### `PATCH /api/admin/users/[id]`
Обновление пользователя

**Body:**
```json
{
  "role": "ADMIN",
  "email_verified": true
}
```

**Защита:**
- Нельзя изменить свою роль

#### `DELETE /api/admin/users/[id]`
Удаление пользователя

**Защита:**
- Нельзя удалить себя

## 🎯 Особенности реализации

### 1. **Пагинация на стороне сервера**
```typescript
const skip = (page - 1) * limit;
const users = await prisma.users.findMany({
  skip,
  take: limit,
  orderBy: { created_at: 'desc' },
});
```

### 2. **Поиск (case-insensitive)**
```typescript
where: {
  OR: [
    { email: { contains: search, mode: 'insensitive' } },
    { name: { contains: search, mode: 'insensitive' } },
  ],
}
```

### 3. **Аватары с инициалами**
```typescript
const getInitials = (email: string, name: string | null) => {
  if (name) {
    return name.split(' ').map(n => n[0]).join('').toUpperCase().slice(0, 2);
  }
  return email.slice(0, 2).toUpperCase();
};
```

### 4. **Сериализация дат**
```typescript
const serializedUsers = users.map(user => ({
  ...user,
  created_at: user.created_at.toISOString(),
  updated_at: user.updated_at.toISOString(),
}));
```

## 🚀 Использование

### Доступ к разделу
```
/admin/users
```

### Навигация
Раздел "Пользователи" в боковом меню админ-панели

### Действия администратора

1. **Просмотр списка**
   - Автоматически загружается при открытии

2. **Поиск пользователя**
   - Введите email или имя в поле поиска
   - Результаты обновляются автоматически

3. **Фильтрация по роли**
   - Выберите "Все роли", "Пользователи" или "Администраторы"

4. **Редактирование**
   - Нажмите иконку карандаша
   - Измените роль или статус email
   - Сохраните изменения

5. **Удаление**
   - Нажмите иконку корзины
   - Подтвердите действие (галочка)
   - Или отмените (крестик)

6. **Пагинация**
   - Используйте стрелки для навигации
   - Показывается текущая страница и общее количество
