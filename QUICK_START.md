## 🔧 Кастомизация

### Изменить цвета
Найдите в `app/page.tsx`:
```tsx
// Замените эти классы
from-emerald-500 to-teal-600  // Основной градиент
emerald-500                    // Основной цвет
teal-600                       // Вторичный цвет
```

### Изменить тексты
Все тексты в `app/page.tsx`:
- Заголовки (h1, h2, h3)
- Описания (p)
- Кнопки (button)
- Списки (features, pricing)

### Добавить секцию
Скопируйте структуру:
```tsx
<section className="py-20 px-4 sm:px-6 lg:px-8">
  <div className="max-w-7xl mx-auto">
    {/* Ваш контент */}
  </div>
</section>
```

## 🗄️ Настройка базы данных

### Если еще не настроили:

1. **Обновите `.env`**:
```env
DATABASE_URL="postgresql://postgres:ВАШ_ПАРОЛЬ@localhost:5432/bite_lens?schema=public"
```

2. **Примените миграции**:
```bash
npm run db:migrate
```

3. **Сгенерируйте Prisma Client**:
```bash
npm run db:generate
```

📖 **Подробнее**: См. [MIGRATION_GUIDE.md](./MIGRATION_GUIDE.md)
