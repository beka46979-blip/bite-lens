# 💳 Управление тарифными планами

## ✅ Статус: РЕАЛИЗОВАНО

Система управления тарифными планами полностью реализована и готова к использованию!

## 📋 Что реализовано

### 1. База данных ✅
**Новая таблица `subscription_plans`:**
- `plan_type` - тип плана (FREE, PRO, PREMIUM) - уникальный
- `name_ru`, `name_en`, `name_kg` - названия на трех языках
- `description_ru`, `description_en`, `description_kg` - описания
- `price_monthly`, `price_yearly` - цены за месяц и год
- `currency` - валюта (USD, EUR, RUB, KGS)
- `daily_ai_analysis_limit` - лимит ИИ-анализа в день (null = безлимит)
- `has_gamification` - включена ли геймификация
- `has_advanced_analytics` - расширенная аналитика
- `has_meal_planning` - планирование питания
- `has_priority_support` - приоритетная поддержка
- `is_active` - активен ли план
- `sort_order` - порядок сортировки

### 2. API маршруты ✅
- **GET `/api/admin/subscription-plans`** - получить все планы
- **POST `/api/admin/subscription-plans`** - создать новый план
- **GET `/api/admin/subscription-plans/[id]`** - получить один план
- **PATCH `/api/admin/subscription-plans/[id]`** - обновить план
- **DELETE `/api/admin/subscription-plans/[id]`** - удалить план

Все маршруты защищены проверкой `getCurrentAdmin()`.

### 3. Админ-панель ✅
**Страницы:**
- `/admin/subscription-plans` - список всех планов с таблицей
- `/admin/subscription-plans/[id]/edit` - редактирование плана

**Функции:**
- ✅ Просмотр всех планов в красивой таблице
- ✅ Статистика: всего планов, активных, средняя цена
- ✅ Редактирование всех параметров плана
- ✅ Визуальные индикаторы функций (галочки/крестики)
- ✅ Цветные бейджи для типов планов
- ✅ Иконка "Безлимит" для неограниченных планов

### 4. Скрипты ✅
**`init-subscription-plans.mjs`** - инициализация планов с дефолтными значениями:
- FREE: $0, 3 фото/день, без дополнительных функций
- PRO: $9.99/мес, безлимит, геймификация + аналитика + планирование
- PREMIUM: $19.99/мес, безлимит, все функции + приоритетная поддержка

## 🚀 Использование

### 1. Инициализация планов
```bash
node init-subscription-plans.mjs
```

Создаст три дефолтных плана:
- **Бесплатный**: $0, 3 фото/день
- **Профессиональный**: $9.99/мес, безлимит, геймификация
- **Премиум**: $19.99/мес, безлимит, все функции

### 2. Доступ к управлению
1. Войдите в админку: http://localhost:3000/admin/login
2. На dashboard нажмите "Управление тарифами"
3. Или перейдите напрямую: http://localhost:3000/admin/subscription-plans

### 3. Редактирование плана
1. В таблице планов нажмите "Редактировать"
2. Измените нужные параметры:
   - **Названия и описания** на трех языках (RU, EN, KG)
   - **Цены** за месяц и год
   - **Лимит ИИ** (пустое поле = безлимит)
   - **Функции** (чекбоксы)
   - **Статус** (активен/неактивен)
3. Нажмите "Сохранить изменения"

## 💡 Возможности системы

### Управление ценами
- Установка цен за месяц и год
- Поддержка разных валют (USD, EUR, RUB, KGS)
- Возможность установить $0 для бесплатного плана

### Лимиты ИИ-анализа
- Установка количества фото в день
- `null` = безлимитный анализ
- Примеры:
  - FREE: 3 фото/день
  - PRO: безлимит
  - PREMIUM: безлимит

### Функции тарифов
**Геймификация (has_gamification):**
- Ударный режим
- Достижения
- Стрики (серии дней)
- Мотивационные уведомления

**Расширенная аналитика (has_advanced_analytics):**
- Детальные графики
- Отчеты по питанию
- Прогнозы достижения целей
- Сравнение периодов

**Планирование питания (has_meal_planning):**
- Составление меню
- Рецепты
- Списки покупок
- Рекомендации

**Приоритетная поддержка (has_priority_support):**
- Быстрые ответы
- Персональный менеджер
- Приоритет в очереди
- Консультации

### Активация/деактивация
- Деактивированные планы скрыты от пользователей
- Существующие подписки продолжают работать
- Можно временно отключить план для обновления

## 📈 Примеры использования

### Изменить цену PRO плана
1. Откройте `/admin/subscription-plans`
2. Найдите план "Профессиональный"
3. Нажмите "Редактировать"
4. Измените `price_monthly` на новое значение
5. Сохраните

### Добавить безлимит для FREE
1. Откройте редактирование FREE плана
2. Очистите поле "Количество фото в день"
3. Сохраните
4. Теперь FREE = безлимит

### Включить геймификацию для FREE
1. Откройте редактирование FREE плана
2. Отметьте чекбокс "Геймификация"
3. Сохраните
4. Теперь FREE пользователи видят достижения

### Деактивировать PREMIUM
1. Откройте редактирование PREMIUM плана
2. Снимите галочку "План активен"
3. Сохраните
4. PREMIUM скрыт от новых пользователей

## 🎯 Интеграция с приложением

### Получение плана пользователя
```typescript
const user = await prisma.user.findUnique({
  where: { id: userId },
  include: {
    subscriptions: {
      where: { status: 'ACTIVE' },
      orderBy: { created_at: 'desc' },
      take: 1,
    },
  },
});

const planType = user.subscriptions[0]?.plan_type || 'FREE';

const plan = await prisma.subscription_plans.findUnique({
  where: { plan_type: planType },
});
```

### Проверка лимита ИИ
```typescript
const plan = await prisma.subscription_plans.findUnique({
  where: { plan_type: user.planType },
});

if (plan.daily_ai_analysis_limit !== null) {
  const todayCount = await prisma.food_snaps.count({
    where: {
      user_id: userId,
      created_at: {
        gte: new Date(new Date().setHours(0, 0, 0, 0)),
      },
    },
  });

  if (todayCount >= plan.daily_ai_analysis_limit) {
    throw new Error('Достигнут дневной лимит анализа');
  }
}
```

### Проверка доступа к функции
```typescript
const plan = await prisma.subscription_plans.findUnique({
  where: { plan_type: user.planType },
});

if (!plan.has_gamification) {
  throw new Error('Геймификация недоступна в вашем тарифе');
}
```
