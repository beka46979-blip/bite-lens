# ➕ Создание и удаление тарифных планов

## ✅ Реализовано

### 1. Создание нового плана ✅
**Страница:** `/admin/subscription-plans/new`

**Функции:**
- Выбор типа плана (FREE, PRO, PREMIUM)
- Заполнение названий на 3 языках
- Заполнение описаний на 3 языках
- Установка цен (месяц/год)
- Настройка лимита ИИ-анализа
- Включение/выключение функций
- Установка порядка сортировки
- Активация/деактивация плана

**Валидация:**
- Проверка уникальности типа плана
- Обязательные поля: тип, названия, описания, цены
- Автоматическая проверка дубликатов

### 2. Удаление плана ✅
**Функция:** Кнопка "Удалить" в таблице планов

**Особенности:**
- Двухэтапное подтверждение
- Индикатор загрузки
- Автоматическое обновление списка
- Защита от случайного удаления

## 🚀 Использование

### Создание нового плана

1. **Откройте страницу управления тарифами:**
   ```
   http://localhost:3000/admin/subscription-plans
   ```

2. **Нажмите кнопку "Создать план"** (розовая кнопка справа вверху)

3. **Заполните форму:**

   **Тип плана:**
   - FREE - Бесплатный
   - PRO - Профессиональный
   - PREMIUM - Премиум
   
   ⚠️ Тип должен быть уникальным!

   **Названия (обязательно):**
   - RU: Бесплатный
   - EN: Free
   - KG: Акысыз

   **Описания (обязательно):**
   - RU: Базовые функции для начала работы
   - EN: Basic features to get started
   - KG: Негизги функциялар

   **Цены:**
   - Месячная: $0.00 - $999.99
   - Годовая: $0.00 - $9999.99

   **Лимит ИИ:**
   - Пустое поле = безлимит
   - Число = количество фото в день

   **Функции (чекбоксы):**
   - ☑️ Геймификация
   - ☑️ Расширенная аналитика
   - ☑️ Планирование питания
   - ☑️ Приоритетная поддержка

   **Порядок сортировки:**
   - 0 = первый в списке
   - 1, 2, 3... = следующие

   **Статус:**
   - ☑️ План активен

4. **Нажмите "Создать план"**

5. **Готово!** План появится в списке

### Удаление плана

1. **Откройте страницу управления тарифами**

2. **Найдите план в таблице**

3. **Нажмите кнопку "Удалить"** (красная кнопка)

4. **Подтвердите удаление:**
   - Появятся кнопки "Подтвердить" и "Отмена"
   - Нажмите "Подтвердить" для удаления
   - Или "Отмена" для отмены

5. **Готово!** План удален из базы данных

## ⚠️ Важные замечания

### При создании плана:

1. **Тип плана уникален**
   - Нельзя создать два плана с одинаковым типом
   - Если план с таким типом существует, появится ошибка
   - Решение: выберите другой тип или удалите существующий

2. **Все поля обязательны**
   - Названия на всех языках
   - Описания на всех языках
   - Цены (можно $0.00)

3. **Лимит ИИ**
   - Пустое поле = безлимит (∞)
   - 0 = нельзя анализировать фото
   - 1+ = количество фото в день

4. **Порядок сортировки**
   - Меньшее число = выше в списке
   - Одинаковые числа = сортировка по дате создания

### При удалении плана:

1. **Удаление необратимо**
   - План удаляется из базы данных навсегда
   - Восстановить нельзя
   - Будьте осторожны!

2. **Существующие подписки**
   - Если у пользователей есть подписки на этот план
   - Они могут перестать работать
   - Рекомендуется сначала деактивировать план

3. **Альтернатива удалению**
   - Вместо удаления можно деактивировать план
   - Снимите галочку "План активен"
   - План скроется от новых пользователей
   - Существующие подписки продолжат работать
