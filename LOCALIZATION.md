# 🌍 Локализация проекта Bite Lens

## Обзор

Проект поддерживает многоязычность с помощью JSON-файлов переводов. Реализована поддержка русского и кыргызского языков.

## 📁 Структура

```
app/i18n/
├── index.ts                    # Главный файл с экспортом переводов
├── types.ts                    # TypeScript типы
├── useTranslation.ts           # React хук для клиентских компонентов
├── README.md                   # Документация по структуре
├── USAGE.md                    # Подробное руководство по использованию
└── locales/
    ├── kg/                     # Кыргызский язык
    │   ├── landing.json        # Переводы главной страницы
    │   └── layout.json         # Переводы метаданных
    └── ru/                     # Русский язык (по умолчанию)
        ├── landing.json
        └── layout.json
```

## 🚀 Быстрый старт

### Использование в компонентах

```tsx
'use client';

import { useTranslation } from '@/app/i18n/useTranslation';

export function MyComponent() {
  const { t, locale, changeLocale } = useTranslation('landing');
  
  return (
    <div>
      <h1>{t.header.title}</h1>
      <p>{t.hero.description}</p>
    </div>
  );
}
```

### Переключатель языков

Компонент `LanguageSwitcher` уже интегрирован в шапку сайта:

```tsx
import { LanguageSwitcher } from '@/app/components/LanguageSwitcher';

<LanguageSwitcher 
  currentLocale={locale} 
  onLocaleChange={changeLocale} 
/>
```

## 📝 Добавление новых переводов

### 1. Добавьте ключи в JSON файлы

**app/i18n/locales/ru/landing.json:**
```json
{
  "newSection": {
    "title": "Новый заголовок"
  }
}
```

**app/i18n/locales/kg/landing.json:**
```json
{
  "newSection": {
    "title": "Жаңы аталыш"
  }
}
```

### 2. Используйте в компоненте

```tsx
const { t } = useTranslation('landing');
<h2>{t.newSection.title}</h2>
```

## 🔧 Технические детали

### Хранение языка

Выбранный язык сохраняется в `localStorage` под ключом `bite-lens-locale`.

### TypeScript типы

Все переводы полностью типизированы для безопасности типов и автодополнения в IDE.

### Структура JSON

Каждый JSON файл организован по секциям страницы:

```json
{
  "header": { ... },
  "hero": { ... },
  "features": { ... },
  "footer": { ... }
}
```
